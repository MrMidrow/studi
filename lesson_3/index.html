селекторы CSS

селекторы css можно поделить на пять категорий:

  • Простые селекторы
  • Селекторы-комбинаторы
  • Селекторы псевдоклассов
  • Сеелекторы псевдоэлементов
  • Селектор атрибутов

Простые селекторы

Выбор элементов на основе

  • имени *
  • идентификатора "id"
  • класса "class"

Выбор по имени *

Селектор элементов выбирает элементы HTML на основе имени элемента


В примере ниже все елементы < p > на странице будут отображаться по центру с зеленым цветом текста

пример

параграф за выбором имени*

p {
text-align: center;
color: green;
}

Выбор по идентификатору "id"

Селектор id использует атрибут id элемента HTML для выбора определенного элемента. Идентификатор элемента уникален на странице, поэтому селектор идентификаторов используется для выбора одного уникального элемента!
Чтобы выбрать элемент с определенным идентификатором, напишите символ решетки (#), за которым следует идентификатор элемента.

Приведенное ниже правило CSS будет применено к HTML-элементу с id="item":

пример

параграф с id="item"

#item {
text-align: center;
color: green;
}


Выбор по классу class

Селектор класса выбирает элементы HTML с определенным атрибутом класса.

Чтобы выбрать элементы с определенным классом, напишите символ точки (.), а затем имя класса. (selector.class{})


Приведенное ниже правило CSS будет применено к HTML-элементу с class="item":

пример

параграф с class="item"

.item {
text-align: center;
color: green;
}


Так же можно его записать в таком порядке

пример

p.item {
text-aling: center;
color: green;
;



У одного HTML-элемента может быть несколько классов
Элементы HTML также могут относиться к более чем одному классу и будут наследовать сразу два стиля

Приведенное ниже правило CSS будет применено к HTML-элементу с class="textstyle styletext"

пример

< р class="item1 item2">этот параграф имеет два класса < /р >

.item1 {
text-align: center;
}
.item2 {
color: green;
}

Группировка селекторов

.h1 {
color: blue;
}
.h2 {
color:blue;
}
.h3 {
color:blue;
}

можно записать через запятую для того что бы минимизировать код

.h1, .h2, .h3 {
color:blue;
}


Универсальный селектор

Универсальный селектор (*) выбирает все элементы HTML на странице.

пример

Приведенное ниже правило CSS повлияет на каждый элемент HTML на странице:

* {
text-align: center;
color: green;
}


Селекторы-комбинаторы

Селектор CSS может содержать более одного простого селектора. Между простыми селекторами мы можем включить комбинатор.

В CSS есть четыре разных комбинатора:

селекторы потомков (используеться пробел)

В следующем примере выбираются все элементы < p > внутри элементов < div >:

пример

.combo1 p {
text-align:right;
color:blue;
}


Дочерний селектор выбераеться с помощью ">"

Селектор дочерних элементов выбирает все элементы, которые являются дочерними элементами указанного элемента

пример

.combo2 > p {
color:red;
}


Соседние селекторы используют знак "+"

Селектор соседних элементов используется для выбора элемента, который находится непосредственно после другого определенного элемента.
Родственные элементы должны иметь один и тот же родительский элемент, а «смежный» означает «непосредственно следующий»

В следующем примере выбирается первый элемент < p >, расположенный сразу после элементов < div >

пример

div + p {
color:red;
}


Общие селекторы(братские/сестринские) используют знак "~"

Общий селектор родственных элементов выбирает все элементы, которые являются следующими одноуровневыми элементами указанного элемента

В следующем примере выбираются все элементы < p>, которые являются следующими одноуровневыми элементами < div>

пример

.combo ~ p {
color:red;
}



Селекторы псевдоклассов

Псевдокласс используется для определения особого состояния элемента

Якорные псевдоклассы

Могут отображаться по разному

пример

a:link {
color:#3333;
}

a:visited {
color:#3333;
}

a:hover {
color:#3333;
}

a:active {
color:#3333;
}


Еще есть один псевдокласс :first-child. Псевдокласс :first-child соответствует указанному элементу, который является первым дочерним элементом другого элемента

пример

p:first-child {
color:red;
}



Селекторы псевдо элементов

Псевдоэлемент CSS используется для стилизации определенных частей элемента

Первый псевдо элемент ::first-line

Псевдоэлемент ::first-line используется для добавления особого стиля к первой строке текста

пример

p::first-line {
color:green;
}

Примечание . ::first-line Псевдоэлемент можно применять только к элементам блочного уровня

Псевдоэлемент ::first-letter

Псевдоэлемент ::first-letterиспользуется для добавления особого стиля к первой букве текста

пример

p::first-letter {
color: green;
}

Примечание . ::first-letter Псевдоэлемент можно применять только к элементам блочного уровня

Псевдоэлемент ::before

Псевдоэлемент ::beforeможно использовать для вставки содержимого перед содержимым элемента

пример

p::before {
content: "";
}

псевдоэлемент ::after

Псевдоэлемент ::afterможно использовать для вставки содержимого после содержимого элемента

пример

p::after {
content:"";
}


Стиль HTML-элементов с определенными атрибутами

Можно стилизовать элементы HTML, которые имеют определенные атрибуты или значения атрибутов.

CSS [атрибут] селектор

Селектор [attribute]используется для выбора элементов с указанным атрибутом

пример

a[attribute] {
color: red;
}

CSS-селектор [attribute="velue"]

Cелектор [attribute="value"]используется для выбора элементов с указанным атрибутом и значением

пример

p[attribute="velue"] {
color: red;
}

CSS [attribute~="velue"] селектор

Селектор [attribute~="value"]используется для выбора элементов со значением атрибута, содержащим указанное слово

пример

[title~="flower"] {
border: 5px solid black;
}

CSS [attribute|="velue"] селектор

Селектор [attribute|="value"]используется для выбора элементов с указанным атрибутом, значение которого может быть точно указанным значением или указанным значением, за которым следует дефис (-).
Примечание . Значение должно быть целым словом, либо отдельно, например, class="top", либо с последующим дефисом (-), например class="top-text"

пример

[class|="top"] {
background: red;
}

CSS [attribute^="velue"] селектор

Селектор [attribute^="value"]используется для выбора элементов с указанным атрибутом, значение которого начинается с указанного значения.
В следующем примере выбираются все элементы со значением атрибута класса, начинающимся с «top»:
Примечание . Значение не обязательно должно быть целым словом!

пример

[class^="value"] {
background: red;
}

CSS-селектор [attribute$="value"]

Селектор [attribute$="value"]используется для выбора элементов, значение атрибута которых заканчивается указанным значением.
В следующем примере выбираются все элементы со значением атрибута класса, оканчивающимся на «тест»:
Примечание . Значение не обязательно должно быть целым словом!

пример

[class$="value"] {
background: red;
}

CSS-селектор [attribute*="value"]

Селектор [attribute*="value"]используется для выбора элементов, значение атрибута которых содержит указанное значением.
В следующем примере выбираются все элементы со значением атрибута класса, содержащим "te":
Примечание . Значение не обязательно должно быть целым словом

пример

[class*="value"] {
background:color;
}
